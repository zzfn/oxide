# AskUserQuestion 工具测试指南

## 功能说明

AskUserQuestion 工具允许 AI 在执行任务时向用户询问问题，支持：
- 单选/多选
- 选项推荐
- 自定义输入

## 测试方法

### 1. 编译并运行 CLI

```bash
cargo build --release
./target/release/oxide
```

### 2. 测试单选

向 AI 提问，让它需要做出选择：

```
你好，我想创建一个新项目，但不确定用什么技术栈。请帮我选择。
```

AI 应该会使用 AskUserQuestion 工具询问你：
- 项目类型（Web/CLI/库等）
- 编程语言偏好
- 其他相关选项

### 3. 测试多选

```
我想为项目添加一些功能，请帮我选择需要的功能。
```

AI 可能会列出多个功能选项，让你选择多个。

### 4. 测试自定义输入

```
我想配置数据库连接，但有特殊需求。
```

AI 可能会提供常见选项，同时允许你输入自定义配置。

## 预期行为

当 AI 调用 AskUserQuestion 时，你会看到交互式选择界面：

**单选模式：**
- 使用 ↑↓ 方向键选择选项
- 推荐选项会显示 ✓ 标记
- 按 Enter 确认选择
- 按 Esc 或 Ctrl+C 取消

**多选模式：**
- 使用 ↑↓ 方向键移动
- 按 Space 键选择/取消选择
- 推荐选项默认已选中
- 按 Enter 确认所有选择
- 如果允许自定义，确认后会询问是否添加自定义选项

## 注意事项

- 按 Ctrl+C 或 Ctrl+D 可以取消问题
- 无效输入会提示重试
- AI 会根据你的回答继续执行任务
