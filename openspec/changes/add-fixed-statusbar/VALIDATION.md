# DECSTBM + Reedline 兼容性验证报告

## 测试程序

已创建 `examples/statusbar_test.rs` - 最小化验证程序

## 运行方式

```bash
cargo run --example statusbar_test
```

## 测试要点

1. **基本功能**
   - 状态栏是否固定在底部
   - 输入内容是否在滚动区正常显示
   - 计数器是否实时更新

2. **关键场景**
   - 输入 `check` - 验证滚动区域保护
   - 输入 `resize` - 测试尺寸变化处理
   - 输入多行 - 验证滚动不影响状态栏

3. **退出清理**
   - Ctrl+D 退出后终端状态是否正常

## 预期结果

✓ 状态栏固定在底部（灰色背景）
✓ 对话内容在上方滚动
✓ 退出后终端恢复正常

## 失败标志

✗ 状态栏随内容向上滚动
✗ Reedline 重置了滚动区域
✗ 退出后终端显示异常

## 下一步

根据测试结果决定：
- 成功 → 实施完整方案
- 失败 → 采用降级方案（仅流式输出时显示状态栏）
